免费功能不会被拿走

流速徽章、当前页面热榜、书签数显示、图片查看增强、长图阅读、内容过滤、Markdown 复制、感谢星图和 Grok 回复辅助仍是免费能力。

Pro license 只用于官方 Chrome 扩展内的高级体验。客户端源码开放，付费价值来自官方构建、自动更新、license 服务、支持和持续维护。

官方版 vs GitHub 社区版

同一套源码按 MIT 协议开放。Chrome Web Store 版本是官方发行版；GitHub 构建、自行加载、油猴脚本和 Safari UserScripts 属于社区/自建路径。

安装和更新 Chrome Web Store 官方版提供一键安装和自动更新。GitHub 社区版需要自行下载 release、开发者模式加载或安装脚本。
Pro 激活 Pro 订阅面向官方 Chrome 扩展，支持 license 激活和状态校验。自建版本可审计和修改源码，但不承诺官方 license 服务支持。
支持范围 官方版包含维护支持和持续发行。社区版按 MIT 协议 as-is 提供，适合开发者审计、二次开发和提交 PR。
脚本版本 Tampermonkey 和 Safari UserScripts 是免费轻量路径，重点覆盖流速徽章和移动端可用性，不等同于完整 Chrome 扩展。

核心能力

流速徽章

每条推文旁显示 views/hour，快速判断内容是否正在起量。

当前页面热榜

浮动面板按流速排序当前可见推文，支持点击回跳。

长图阅读

X 灯箱内支持滚轮缩放、拖拽平移和长图纵向阅读。

内容过滤

本地规则过滤 spam、色情和电报漏斗，规则可远程更新。

隐私边界

扩展从你正在访问的 x.com / pro.x.com 页面和 X 自身响应中读取数据。功能计算在浏览器本地完成，不加入统计 SDK，不放追踪像素。
